# Capacitación React - GoodApps

## Ejecutar el servidor local

```sh
pnpm run dev
```

## Instalación de dependencias

Este comando se utiliza solo cuando necesitas instalar alguna libreria

### Ejemplo

```sh
pnpm install axios
```

Si todavía no tenés ninguna dependencia instalada, primero ejecuta:

```sh
pnpm install
```

Este comando instala todo lo definido en el `package.json`.
Podés confirmar que no hay nada instalado si la carpeta `node_modules` no existe.
